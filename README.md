## Overview

Game Stack utilizes two APIs (RAWG and Google Fonts) to showcase thousands of games and allow a user to dynamically interact with the webpage by saving their favorite games. The project encompasses HTML, CSS, Javascript, and a local server.  

## API Documentation

[RAWG API](https://rawg.io/apidocs): Online Video Game Database
[Google Fonts API](https://developers.google.com/fonts): Simple to Use Web Fonts
